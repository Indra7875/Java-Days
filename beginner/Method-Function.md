### Method / Function 
* It is a set of instruction to do a perticular task.
* Method/Function are used to avoid duplicate number of lines in program.
### Method Header
![MethodHeader]()

* Method Header consist of 
1) Modifier
2) Return type
3) Method name
4) Formal arguments (datatype variable_name)

* Method Header consist of
1) Method name
